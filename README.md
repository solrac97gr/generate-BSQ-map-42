# generate-BSQ-map-42

Map generator for the BSQ program.

## Compile:
```
make
```

## Use
row = 10
columns = 10
probability_obstacles = 2
```
./generate_map 10 10 2
```

## Generated map

```

..x.......
..........
.x.x......
..........
.x........
.....x....
x.x..x....
.......x..
..........
```
